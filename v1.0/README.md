# 安裝說明
## 安裝方式
* 在powershell下執行
  * UTF-8 BOM編碼
```
>01_install_aria2.ps1
```
* 使用bat檔安裝
  * ANSI編碼
```
C:\Windows\System32\WindowsPowerShell\v1.0\powershell.exe -executionpolicy remotesigned -File 01_install_aria2.ps1 -f
PAUSE
```
## 必要安裝
* 01_install_aria2.ps1
* 02_install_7zip.ps1
## 參數
```
01_install_aria2.ps1 -h
```
