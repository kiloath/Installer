# install_git

1. 基本資料
   ```
   git config --global user.name Kiloath
   git config --global user.email kiloath@outlook.com
   ```
1. 編輯器設定為 notepad++
   ```
   git config --global core.editor $("'$HOME/KiloathApp/Notepad++/notepad++.exe' -multiInst -notabbar -nosession -noPlugin" -replace '\\','/')
   ```
